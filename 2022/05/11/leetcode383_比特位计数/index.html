leetcode383_比特位计数

发表于2022-05-11|更新于2022-05-11|刷题
|字数总计:269|阅读时长:1分钟|阅读量:

题目：给你一个整数 n ，对于 0 <= i <= n 中的每个 i ，计算其二进制表示中 1 的个数 ，返回一个长度为 n + 1 的数组 ans 作为答案。

进阶：设计时间复杂度为O(n)的算法

题解

个人做法：

思路

判断二进制位的1位：二进制与1相与，并且每次右移一位，重复与1相与，若相与结果不为0，则说明该位为1

将1..n的数都转成二进制，进行上述的二进制1位的判断

复杂度

时间复杂度：o(nlogn)，空间复杂度：O(n)

优化方法

思路：找规律，动态规划

状态转移方程：

i 为奇数：

dp[i] = dp[i - 1] + 1

i 为偶数：

dp[i] = dp[i / 2]
