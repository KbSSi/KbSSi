打家劫舍3

发表于2022-05-14|更新于2022-05-15|刷题
|字数总计:347|阅读时长:1分钟|阅读量:

题目：

打家劫舍3，房子构成一个树形，不能连续抢劫树上相邻的节点（树型dp）

实例：

输入: root = [3,2,3,null,3,null,1]
输出: 7
解释: 小偷一晚能够盗取的最高金额 3 + 3 + 1 = 7

题解

个人做法

思路

可以把树转成数组的形式，最后数组间相邻的规则符合树相邻规则，然后按照数组的dp来做吗?

发现前序、中序、后续、层次遍历都不符合，这种想法行不通

官方题解

思路

为每一个节点定义两种状态，抢或者不抢

对于根节点来说，若抢根节点，则不能抢其左孩子和右孩子；若不抢根节点，则既可以抢其右孩子和左孩子，也可以不抢其右孩子和左孩子

因为求根节点的状态前需要知道左孩子和右孩子的节点，则符合树的后续遍历

对两种状态分别求状态转移方程
